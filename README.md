# 🚀 AI VAPT Zero-Day Detection Tool

An AI-powered VAPT framework that automates:
- Crawling
- Fuzzing
- Anomaly detection (ML-based)
- OSINT scanning
- Zero-day pattern detection

---

## ⚙️ Setup


python3 -m venv vapt

source vapt/bin/activate

pip install -r requirements.txt
